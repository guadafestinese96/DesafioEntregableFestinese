# Proyecto en Python – Curso Coderhouse

Este proyecto fue desarrollado como parte del curso de **Python en Coderhouse**.  
Su objetivo principal es aplicar conceptos fundamentales del lenguaje y resolver desafíos prácticos.

## Tecnologías
- Lenguaje: Python

## Funcionalidades
- Ejercicios de lógica y estructuras de control
- Manejo de funciones y modularización
- Uso de listas, diccionarios y estructuras de datos
- Implementación de pequeños programas y desafíos prácticos

## Objetivo académico
- Practicar fundamentos de programación en Python
- Resolver problemas aplicando buenas prácticas
- Preparar bases para proyectos más complejos (automatizaciones, dashboards, IA)

## Cómo ejecutar
1. Clonar el repositorio:
   git clone https://github.com/guadafestinese96/EntregableGuadalupeFestinese.git

2. Entrar a la carpeta del proyecto:
  cd EntregableGuadalupeFestinese
   
3. Ejecutar el archivo principal
  python main.py
